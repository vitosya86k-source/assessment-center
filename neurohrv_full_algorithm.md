### RD — Readiness (Готовность)

**Расшифровка:** Комплексная готовность к физическим и умственным нагрузкам

**Единицы измерения:** % (0-100)

**Что показывает:** Интегральная оценка способности нервной системы адаптироваться к стрессовым воздействиям. Учитывает все ключевые компоненты HRV.

**Формула расчёта:**
```python
def calculate_readiness(rmssd, sdnn, stress_index, hf_power, lf_hf_ratio):
    """
    Расчёт комплексной готовности
    """
    # Нормализация каждого компонента к шкале 0-100
    rmssd_norm = min(rmssd / 50 * 100, 100)  # 50 мс = 100%
    sdnn_norm = min(sdnn / 100 * 100, 100)   # 100 мс = 100%
    si_norm = max(0, 100 - (stress_index / 5))  # SI 500 = 0%
    hf_norm = min(hf_power / 500 * 100, 100)  # 500 ms² = 100%
    
    # LF/HF баланс: оптимум 0.5-2.0
    if 0.5 <= lf_hf_ratio <= 2.0:
        balance_norm = 100
    elif lf_hf_ratio < 0.5:
        balance_norm = lf_hf_ratio / 0.5 * 80 + 20
    else:  # > 2.0
        balance_norm = max(0, 100 - (lf_hf_ratio - 2) * 20)
    
    # Взвешенная сумма
    readiness = (
        0.30 * rmssd_norm +
        0.25 * sdnn_norm +
        0.20 * si_norm +
        0.15 * hf_norm +
        0.10 * balance_norm
    )
    
    return round(readiness)
```

|----------|---------|------|---------------|
| 80-100% | Отличный | 🟢 #00d4aa | Оптимальное состояние. Готов к сложным задачам, принятию решений и лидерству. |
| 60-79% | Хороший | 🟢 #a8e063 | Хорошие ресурсы. Подходит для стандартных задач, может выступать ментором. |
| 50-59% | Норма | 🟡 #ffd93d | Достаточные ресурсы. Рекомендуются рутинные задачи, избегать дополнительного стресса. |

### SR — Stress Resistance (Стрессоустойчивость)

**Расшифровка:** Устойчивость к стрессовым воздействиям

**Единицы измерения:** % (0-100), рассчитывается из Stress Index

**Что показывает:** Способность сохранять функциональное состояние и эффективность при воздействии стрессоров. Основан на индексе напряжения по Баевскому.

**Формула расчёта Stress Index (по Баевскому):**
```python
def calculate_stress_index(rr_intervals):
    """
    SI = AMo / (2 × Mo × MxDMn)
    
    AMo — амплитуда моды (% интервалов в модальном классе)
    Mo — мода (наиболее частое значение RR)
    MxDMn — вариационный размах (max RR - min RR)
    """
    # Построение гистограммы с шагом 50 мс
    bin_width = 50  # мс
    hist, bin_edges = np.histogram(rr_intervals, 
                                    bins=range(int(min(rr_intervals)), 
                                              int(max(rr_intervals)) + bin_width, 
                                              bin_width))
    
    # Мода — центр бина с максимальной частотой
    mode_idx = np.argmax(hist)
    Mo = (bin_edges[mode_idx] + bin_edges[mode_idx + 1]) / 2
    
    # Амплитуда моды — процент интервалов в модальном классе
    AMo = hist[mode_idx] / len(rr_intervals) * 100
    
    # Вариационный размах
    MxDMn = max(rr_intervals) - min(rr_intervals)
    
    # Stress Index
    if Mo > 0 and MxDMn > 0:
        SI = AMo / (2 * Mo / 1000 * MxDMn / 1000)  # переводим в секунды
    else:
        SI = 0
    
    return SI

def stress_index_to_resistance(si):
    """
    Конвертация SI в процент стрессоустойчивости (инверсия)
    """
    if si <= 50:
        return 100
    elif si <= 150:
        return 100 - (si - 50) * 0.25  # 50→100%, 150→75%
    elif si <= 300:
        return 75 - (si - 150) * 0.33  # 150→75%, 300→25%
    elif si <= 500:
        return 25 - (si - 300) * 0.10  # 300→25%, 500→5%
    else:
        return max(0, 5 - (si - 500) * 0.01)
```

### AD — Adaptability (Адаптивность)

**Расшифровка:** Общая адаптационная способность организма

**Единицы измерения:** % (0-100), рассчитывается из SDNN (мс)

**Что показывает:** Способность сердечно-сосудистой системы адаптироваться к изменяющимся условиям. Включает все частотные компоненты вариабельности.

**Формула расчёта SDNN:**
```python
def calculate_sdnn(rr_intervals):
    """
    SDNN = стандартное отклонение всех NN (нормальных RR) интервалов
    
    SDNN = √(Σ(RRᵢ - mean_RR)² / (n-1))
    """
    mean_rr = np.mean(rr_intervals)
    sdnn = np.sqrt(np.sum((rr_intervals - mean_rr) ** 2) / (len(rr_intervals) - 1))
    return sdnn

def sdnn_to_adaptability(sdnn):
    """
    Конвертация SDNN в процент адаптивности
    """
    if sdnn >= 100:
        return 100
    elif sdnn >= 50:
        return 50 + (sdnn - 50) * 1.0  # 50→50%, 100→100%
    elif sdnn >= 30:
        return 30 + (sdnn - 30) * 1.0  # 30→30%, 50→50%
    elif sdnn >= 20:
        return 15 + (sdnn - 20) * 1.5  # 20→15%, 30→30%
    else:
        return max(0, sdnn * 0.75)  # 0→0%, 20→15%
```

### FL — Flexibility (Гибкость НС)

**Расшифровка:** Гибкость нервной системы

**Единицы измерения:** % (0-100), рассчитывается из pNN50 (%) и SD1 (мс)

**Что показывает:** Способность к быстрой адаптации и переключению между задачами. Специфический маркер вагусной (парасимпатической) активности.

**Формула расчёта:**
```python
def calculate_pnn50(rr_intervals):
    """
    pNN50 = процент смежных RR интервалов, различающихся >50 мс
    """
    differences = np.abs(np.diff(rr_intervals))
    count_over_50 = np.sum(differences > 50)
    pnn50 = (count_over_50 / len(differences)) * 100
    return pnn50

def calculate_sd1(rr_intervals):
    """
    SD1 = стандартное отклонение точек от линии идентичности на диаграмме Пуанкаре
    SD1 = √(0.5 × RMSSD²)
    """
    rmssd = calculate_rmssd(rr_intervals)
    sd1 = np.sqrt(0.5 * rmssd ** 2)
    return sd1

def flexibility_score(pnn50, sd1):
    """
    Комбинированный показатель гибкости
    """
    # Нормализация pNN50: 0% = 0, 20%+ = 100
    pnn50_norm = min(pnn50 / 20 * 100, 100)
    
    # Нормализация SD1: 0 мс = 0, 50+ мс = 100
    sd1_norm = min(sd1 / 50 * 100, 100)
    
    # Взвешенная сумма
    flexibility = 0.6 * pnn50_norm + 0.4 * sd1_norm
    return round(flexibility)
```

| 10-15% | 25-40 мс | 60-80% | Хороший | 🟢 | Хорошая гибкость. Успешно переключается, умеренное время адаптации. |

| <1% | <4 мс | <10% | Критический | 🔴 | Критически низкая. Возможна автономная дисфункция, рекомендуется обследование. |

### RC — Recovery (Восстановление)

**Расшифровка:** Способность к восстановлению

**Единицы измерения:** % (0-100), рассчитывается из RMSSD (мс) и HF power (мс²)

**Что показывает:** Активность парасимпатической НС, отвечающей за восстановление и расслабление. Способность быстро восполнять ресурсы между задачами.

**Формула расчёта:**
```python
def calculate_rmssd(rr_intervals):
    """
    RMSSD = √(Σ(RRᵢ₊₁ - RRᵢ)² / (n-1))
    Среднеквадратичное отклонение последовательных разностей
    """
    differences = np.diff(rr_intervals)
    rmssd = np.sqrt(np.mean(differences ** 2))
    return rmssd

def calculate_hf_power(rr_intervals, fs=4):
    """
    HF power = мощность спектра в диапазоне 0.15-0.4 Гц
    Требует интерполяции RR к равномерной сетке
    """
    # Интерполяция к равномерной частоте дискретизации
    time_original = np.cumsum(rr_intervals) / 1000  # в секунды
    time_interp = np.arange(0, time_original[-1], 1/fs)
    rr_interp = np.interp(time_interp, time_original, rr_intervals)
    
    # Удаление тренда
    rr_detrended = rr_interp - np.mean(rr_interp)
    
    # Спектральный анализ (метод Уэлча)
    freqs, psd = signal.welch(rr_detrended, fs=fs, nperseg=min(256, len(rr_detrended)))
    
    # HF диапазон: 0.15-0.4 Гц
    hf_mask = (freqs >= 0.15) & (freqs <= 0.4)
    hf_power = np.trapz(psd[hf_mask], freqs[hf_mask])
    
    return hf_power

def recovery_score(rmssd, hf_power):
    """
    Комбинированный показатель восстановления
    """
    # Нормализация RMSSD: 0 мс = 0, 60+ мс = 100
    rmssd_norm = min(rmssd / 60 * 100, 100)
    
    # Нормализация HF: 0 ms² = 0, 1000+ ms² = 100
    hf_norm = min(hf_power / 1000 * 100, 100)
    
    # Взвешенная сумма
    recovery = 0.5 * rmssd_norm + 0.5 * hf_norm
    return round(recovery)
```

**Диапазоны:**

| RMSSD | HF power | RC% | Уровень | Цвет | Интерпретация |
|-------|----------|-----|---------|------|---------------|
| >50 мс | >500 мс² | 80-100% | Отличный | 🟢 | Отличное восстановление. Быстро восполняет ресурсы, высокая работоспособность. |
| 35-50 мс | 300-500 мс² | 60-80% | Хороший | 🟢 | Хорошее восстановление. Достаточно коротких перерывов для восполнения сил. |
| 20-35 мс | 150-300 мс² | 40-60% | Норма | 🟡 | Достаточное восстановление. Нужны регулярные перерывы для поддержания эффективности. |
| 12-20 мс | 75-150 мс² | 20-40% | Сниженный | 🟠 | Замедленное восстановление. Требуются более длительные паузы, контроль нагрузки. |
| 6-12 мс | 30-75 мс² | 10-20% | Низкий | 🔴 | Низкое восстановление. Высокий риск истощения, нужен полноценный отдых. |
| <6 мс | <30 мс² | <10% | Критический | 🔴 | Критически низкое. Ресурсы истощены, требуется длительное восстановление. |

### EN — Endurance (Выносливость)

**Расшифровка:** Устойчивость к длительным нагрузкам

**Единицы измерения:** % (0-100), рассчитывается из SD2 (мс)

**Что показывает:** Долгосрочная вариабельность, способность поддерживать производительность в течение длительного времени без значимого снижения эффективности.

**Формула расчёта:**
```python
def calculate_sd2(rr_intervals):
    """
    SD2 = стандартное отклонение точек вдоль линии идентичности на диаграмме Пуанкаре
    SD2 = √(2×SDNN² - 0.5×RMSSD²)
    """
    sdnn = calculate_sdnn(rr_intervals)
    rmssd = calculate_rmssd(rr_intervals)
    
    sd2_squared = 2 * sdnn**2 - 0.5 * rmssd**2
    sd2 = np.sqrt(max(0, sd2_squared))  # защита от отрицательных значений
    return sd2

def endurance_score(sd2):
    """
    Конвертация SD2 в процент выносливости
    """
    if sd2 >= 100:
        return 100
    elif sd2 >= 70:
        return 70 + (sd2 - 70) * 1.0  # 70→70%, 100→100%
    elif sd2 >= 50:
        return 50 + (sd2 - 50) * 1.0  # 50→50%, 70→70%
    elif sd2 >= 30:
        return 30 + (sd2 - 30) * 1.0  # 30→30%, 50→50%
    else:
        return max(0, sd2)  # 0→0%, 30→30%
```

| SD2 (мс) | EN% | Уровень | Цвет | Интерпретация |
|----------|-----|---------|------|---------------|
| >80 | 80-100% | Высокий | 🟢 | Высокая выносливость. Способен эффективно работать весь день без значимой усталости. |
| 60-80 | 60-80% | Хороший | 🟢 | Хорошая выносливость. Стабильная работоспособность при стандартном графике. |
| 45-60 | 45-60% | Норма | 🟡 | Достаточная выносливость. К концу дня возможно снижение продуктивности. |
| 30-45 | 30-45% | Сниженный | 🟠 | Ограниченная выносливость. Рекомендуются более частые перерывы. |
| 15-30 | 15-30% | Низкий | 🔴 | Низкая выносливость. Быстрое истощение, короткие рабочие сессии. |
| <15 | <15% | Критический | 🔴 | Критически низкая. Выполнение только кратковременных простых задач. |

### BL — Balance (Вегетативный баланс)

**Расшифровка:** Симпато-вагальный баланс

**Единицы измерения:** % (0-100), рассчитывается из LF/HF ratio

**Что показывает:** Соотношение активности симпатической и парасимпатической НС. Определяет оптимальность состояния для разных типов задач.

**Формула расчёта:**
```python
def calculate_lf_hf_ratio(rr_intervals, fs=4):
    """
    LF/HF = отношение мощности низких частот к высоким
    LF: 0.04-0.15 Гц
    HF: 0.15-0.4 Гц
    """
    # Интерполяция и спектральный анализ (как в HF power)
    time_original = np.cumsum(rr_intervals) / 1000
    time_interp = np.arange(0, time_original[-1], 1/fs)
    rr_interp = np.interp(time_interp, time_original, rr_intervals)
    rr_detrended = rr_interp - np.mean(rr_interp)
    
    freqs, psd = signal.welch(rr_detrended, fs=fs, nperseg=min(256, len(rr_detrended)))
    
    # LF диапазон: 0.04-0.15 Гц
    lf_mask = (freqs >= 0.04) & (freqs <= 0.15)
    lf_power = np.trapz(psd[lf_mask], freqs[lf_mask])
    
    # HF диапазон: 0.15-0.4 Гц
    hf_mask = (freqs >= 0.15) & (freqs <= 0.4)
    hf_power = np.trapz(psd[hf_mask], freqs[hf_mask])
    
    # Отношение
    if hf_power > 0:
        lf_hf_ratio = lf_power / hf_power
    else:
        lf_hf_ratio = 0
    
    return lf_hf_ratio, lf_power, hf_power

def balance_score(lf_hf_ratio):
    """
    Конвертация LF/HF в процент баланса
    Оптимум: 0.5-2.0 = 100%
    """
    if 0.5 <= lf_hf_ratio <= 2.0:
        return 100
    elif lf_hf_ratio < 0.5:
        # Парасимпатическое преобладание
        return max(50, 50 + lf_hf_ratio * 100)  # 0→50%, 0.5→100%
    elif lf_hf_ratio <= 4.0:
        # Умеренное симпатическое преобладание
        return max(40, 100 - (lf_hf_ratio - 2) * 30)  # 2→100%, 4→40%
    else:
        # Выраженное симпатическое преобладание
        return max(0, 40 - (lf_hf_ratio - 4) * 10)  # 4→40%, 8→0%
```

**Диапазоны:**

| LF/HF | BL% | Состояние | Цвет | Интерпретация |
|-------|-----|-----------|------|---------------|
| 0.5-2.0 | 90-100% | Оптимальный баланс | 🟢 | Гармоничное состояние. Оптимально для большинства задач, хорошая адаптация. |
| 0.2-0.5 | 60-90% | ПНС преобладает | 🟢 | Парасимпатика доминирует. Подходит для творческих задач, брейнштормов, обучения. |
| 2.0-3.5 | 55-90% | СНС умеренно | 🟡 | Умеренная активация. Подходит для задач, требующих концентрации и быстрых решений. |
| 3.5-5.0 | 30-55% | СНС выражено | 🟠 | Выраженная активация. Эффективен в дедлайнах, но риск истощения. |
| <0.2 | 40-60% | ПНС избыточно | 🟡 | Глубокое расслабление. Возможна сниженная мобилизация, сонливость. |
| >5.0 | <30% | Стресс | 🔴 | Стрессовая реакция. Снижена эффективность, высокий риск ошибок. |

### SI — Stress Index (Индекс напряжения)

**Расшифровка:** Индекс напряжения регуляторных систем по Баевскому

**Единицы измерения:** условные единицы (у.е.)

**Что показывает:** Степень напряжения регуляторных механизмов, активность симпатической НС и центрального контура регуляции.

### RR — Respiratory Rate (Частота дыхания)

**Расшифровка:** Частота дыхания

**Единицы измерения:** вд/мин (вдохов в минуту)

**Что показывает:** Состояние дыхательного центра, уровень тревожности и общей активации.

**Формула:**
```python
def estimate_respiratory_rate(rr_intervals, fs=4):
    """
    Оценка частоты дыхания через пик HF-диапазона
    HF-колебания синхронизированы с дыханием
    """
    # Спектральный анализ
    time_original = np.cumsum(rr_intervals) / 1000
    time_interp = np.arange(0, time_original[-1], 1/fs)
    rr_interp = np.interp(time_interp, time_original, rr_intervals)
    rr_detrended = rr_interp - np.mean(rr_interp)
    
    freqs, psd = signal.welch(rr_detrended, fs=fs, nperseg=min(256, len(rr_detrended)))
    
    # Находим пик в HF диапазоне (0.15-0.4 Гц)
    hf_mask = (freqs >= 0.15) & (freqs <= 0.4)
    hf_freqs = freqs[hf_mask]
    hf_psd = psd[hf_mask]
    
    if len(hf_psd) > 0:
        peak_idx = np.argmax(hf_psd)
        peak_freq = hf_freqs[peak_idx]
        respiratory_rate = peak_freq * 60  # Гц → вд/мин
    else:
        respiratory_rate = 15  # среднее значение по умолчанию
    
    return round(respiratory_rate, 1)
```

**Диапазоны:**

| RR | Индикатор | Состояние | Интерпретация |
|----|-----------|-----------|---------------|
| <10 | 🟡 | Очень редкое | Глубокое расслабление или медитация |
| 10-12 | 🟢 | Редкое | Спокойное, расслабленное состояние |
| 12-16 | 🟢 | Норма | Нормальное рабочее состояние |
| 16-20 | 🟡 | Учащённое | Умеренная активация или лёгкое волнение |
| >20 | 🟠 | Частое | Тревога, стресс или физическая нагрузка |

### BA — Biological Age (Биологический возраст)

**Расшифровка:** Биологический возраст по данным HRV

**Единицы измерения:** годы

**Что показывает:** Функциональный возраст сердечно-сосудистой системы, отражающий общее состояние здоровья и адаптационные резервы.

**Формула (упрощённая, на основе RMSSD):**
```python
def calculate_biological_age(rmssd, sdnn, stress_index):
    """
    Оценка биологического возраста на основе HRV показателей
    Основана на регрессионных моделях Kubios и исследованиях
    """
    # Базовая оценка через RMSSD (основной предиктор)
    # RMSSD снижается примерно на 1-2 мс каждые 10 лет
    # Референсные значения: 20 лет ≈ 50 мс, 60 лет ≈ 20 мс
    
    if rmssd >= 50:
        base_age = 20
    elif rmssd >= 40:
        base_age = 20 + (50 - rmssd) * 1.0  # 50→20, 40→30
    elif rmssd >= 30:
        base_age = 30 + (40 - rmssd) * 1.5  # 40→30, 30→45
    elif rmssd >= 20:
        base_age = 45 + (30 - rmssd) * 1.5  # 30→45, 20→60
    elif rmssd >= 10:
        base_age = 60 + (20 - rmssd) * 1.5  # 20→60, 10→75
    else:
        base_age = 75 + (10 - rmssd) * 2.0
    
    # Коррекция по SDNN
    if sdnn > 100:
        base_age -= 5
    elif sdnn < 30:
        base_age += 5
    
    # Коррекция по Stress Index
    if stress_index < 100:
        base_age -= 3
    elif stress_index > 300:
        base_age += 5
    
    # Ограничение диапазона
    biological_age = max(18, min(90, round(base_age)))
    
    return biological_age
```

**Отображение:**
- Показывается числовое значение: "Биологический возраст: 32 года"
- БЕЗ сравнения с хронологическим возрастом (мы его не знаем)
- Дополнительно указывается источник: "По данным HRV (алгоритм Kubios)"

- Аэробные нагрузки умеренной интенсивности 3-4 раза в неделю (ходьба, плавание, велосипед).
- Диета с достаточным количеством омега-3, магния, витаминов группы B.
- Регулярный отпуск и выходные без работы.

---

### Для низкой адаптивности (AD <50%)

**Quick wins:**
- При изменении условий — пауза 2-3 минуты на осмысление.
- Записать новую информацию, структурировать.
- Попросить дополнительное время на адаптацию.

**Среднесрочные:**
- Постепенное введение изменений (не всё сразу).
- Создание чек-листов для новых процессов.
- Регулярный выход из зоны комфорта в безопасных условиях.

**Долгосрочные:**
- Развитие стрессоустойчивости (см. рекомендации выше).
- Физическая активность для улучшения общей вариабельности.
- При SDNN <30 мс — консультация с кардиологом.

---

### Для низкой выносливости (EN <50%)

**Quick wins:**
- Планировать сложные задачи на пик энергии (обычно утро).
- Разбивать большие задачи на этапы по 30-45 минут.
- Гидратация: пить воду регулярно.

**Среднесрочные:**
- Оптимизация питания: избегать тяжёлой пищи в обед, стабильный уровень сахара.
- Лёгкая разминка в середине дня.
- Снижение общей нагрузки, делегирование.

**Долгосрочные:**
- Тренировка выносливости: регулярные аэробные нагрузки с постепенным увеличением длительности.
- Работа над качеством сна.
- Проверка уровня железа, витамина D, функции щитовидной железы.

---

### Для низкого баланса (BL <50%)

**При преобладании СНС (LF/HF >3):**

*Quick wins:*
- Активация парасимпатики через дыхание: выдох длиннее вдоха (4-8 сек).
- Массаж воротниковой зоны 2-3 минуты.
- Прослушивание спокойной музыки.

*Среднесрочные:*
- Снижение стимуляторов (кофе, энергетики).
- Вечерний ритуал расслабления.
- Ограничение новостей и соцсетей.

**При преобладании ПНС (LF/HF <0.3):**

*Quick wins:*
- Короткая интенсивная активность (20 прыжков, быстрая ходьба).
- Холодный душ или умывание холодной водой.
- Яркий свет, открыть окно.

*Среднесрочные:*
- Регулярная физическая активность.
- Проверка режима сна (возможно, избыток сна).
- Контроль уровня сахара, исключение гипотиреоза.

```python
def generate_risks(metrics):

        risks.append({

        })

    if metrics['sdnn'] < 20:
        risks.append({

        })
    elif metrics['sdnn'] < 30:
        risks.append({
            'level': 'warning',
            'text': f"SDNN {metrics['sdnn']:.0f} мс (порог 30) — сниженная вариабельность. Ограниченные адаптационные резервы."
        })
    
    # RMSSD
    if metrics['rmssd'] < 6:
        risks.append({
            'level': 'critical',
            'text': f"RMSSD {metrics['rmssd']:.0f} мс (порог 6) — критически низкая парасимпатическая активность. Риск истощения."
        })
    elif metrics['rmssd'] < 12:
        risks.append({
            'level': 'warning',
            'text': f"RMSSD {metrics['rmssd']:.0f} мс (порог 12) — сниженная восстановительная способность."
        })
    
    # HR
    if metrics['hr'] > 100:
        risks.append({
            'level': 'warning',
            'text': f"HR {metrics['hr']:.0f} уд/мин (порог 100) — тахикардия. Возможно, стресс или физическая нагрузка."
        })
    elif metrics['hr'] < 50 and metrics['rmssd'] < 20:
        risks.append({
            'level': 'warning',
            'text': f"HR {metrics['hr']:.0f} уд/мин при низком RMSSD — возможна патологическая брадикардия."
        })
    
    return risks
```

## 3.1 Детекция всплесков (вверх и вниз)

```python
def detect_events(data, window_size=30, hr_threshold=0.12, si_threshold=0.30):
    """
    Определение событий — отклонений от скользящей средней
    
    Параметры:
    - window_size: размер окна для скользящей средней (секунды)
    - hr_threshold: порог отклонения HR (12% = 0.12)
    - si_threshold: порог отклонения SI (30% = 0.30)
    
    Возвращает список событий с типом (spike_up, spike_down)
    """
    events = []
    
    hr_data = data['Heart_Rate_bpm'].values
    si_data = data['Approximate_Stress_Index'].values
    timestamps = data['Timestamp_ISO'].values
    
    for i in range(window_size, len(hr_data)):
        # Скользящие средние
        mean_hr = np.mean(hr_data[i-window_size:i])
        mean_si = np.mean(si_data[i-window_size:i])
        
        # Отклонения (относительные)
        hr_deviation = (hr_data[i] - mean_hr) / mean_hr if mean_hr > 0 else 0
        si_deviation = (si_data[i] - mean_si) / mean_si if mean_si > 0 else 0
        
        # Всплеск ВВЕРХ (активация, стресс)
        if hr_deviation > hr_threshold or si_deviation > si_threshold:
            events.append({
                'index': i,
                'timestamp': timestamps[i],
                'type': 'spike_up',
                'hr_before': mean_hr,
                'hr_current': hr_data[i],
                'hr_change_percent': hr_deviation * 100,
                'si_before': mean_si,
                'si_current': si_data[i],
                'si_change_percent': si_deviation * 100
            })
        
        # Всплеск ВНИЗ (резкое снижение, возможно расслабление или провал)
        elif hr_deviation < -hr_threshold or si_deviation < -si_threshold:
            events.append({
                'index': i,
                'timestamp': timestamps[i],
                'type': 'spike_down',
                'hr_before': mean_hr,
                'hr_current': hr_data[i],
                'hr_change_percent': hr_deviation * 100,
                'si_before': mean_si,
                'si_current': si_data[i],
                'si_change_percent': si_deviation * 100
            })
    
    # Объединение последовательных событий одного типа
    merged_events = merge_events(events)
    
    return merged_events

def merge_events(events, gap_threshold=5):
    """
    Объединяет последовательные события одного типа
    gap_threshold: максимальный разрыв между событиями для объединения (секунды)
    """
    if not events:
        return []
    
    merged = []
    current = events[0].copy()
    current['start_index'] = current['index']
    current['end_index'] = current['index']
    
    for event in events[1:]:
        if (event['type'] == current['type'] and 
            event['index'] - current['end_index'] <= gap_threshold):
            # Продлеваем текущее событие
            current['end_index'] = event['index']
            current['hr_current'] = max(current['hr_current'], event['hr_current']) \
                if event['type'] == 'spike_up' else min(current['hr_current'], event['hr_current'])
            current['si_current'] = max(current['si_current'], event['si_current']) \
                if event['type'] == 'spike_up' else min(current['si_current'], event['si_current'])
        else:
            # Завершаем текущее, начинаем новое
            current['duration'] = current['end_index'] - current['start_index'] + 1
            merged.append(current)
            current = event.copy()
            current['start_index'] = current['index']
            current['end_index'] = current['index']
    
    # Добавляем последнее событие
    current['duration'] = current['end_index'] - current['start_index'] + 1
    merged.append(current)
    
    return merged

def classify_event(event):
    """
    Классификация события по интенсивности и типу
    """
    if event['type'] == 'spike_up':
        hr_change = event['hr_change_percent']
        si_change = event['si_change_percent']
        
        if hr_change > 20 or si_change > 50:
            return {
                'severity': 'high',
                'name': 'Пиковая активация',
                'description': 'Максимальное напряжение. Выраженная стресс-реакция или интенсивная мобилизация.'
            }
        elif hr_change > 15 or si_change > 35:
            return {
                'severity': 'medium',
                'name': 'Выраженная активация',
                'description': 'Значительная мобилизация ресурсов. Реакция на стрессор или сложную задачу.'
            }
        else:
            return {
                'severity': 'low',
                'name': 'Умеренная активация',
                'description': 'Небольшое повышение активации. Нормальная реакция на изменение условий.'
            }
    
    else:  # spike_down
        hr_change = abs(event['hr_change_percent'])
        si_change = abs(event['si_change_percent'])
        
        if hr_change > 20 or si_change > 50:
            return {
                'severity': 'high',
                'name': 'Резкое снижение',
                'description': 'Выраженное падение активации. Возможно утомление или переход в фазу расслабления.'
            }
        elif hr_change > 15 or si_change > 35:
            return {
                'severity': 'medium',
                'name': 'Заметное снижение',
                'description': 'Значительное снижение тонуса. Расслабление или начало утомления.'
            }
        else:
            return {
                'severity': 'low',
                'name': 'Умеренное снижение',
                'description': 'Небольшое снижение активации. Нормальная вариация состояния.'
            }
```

```python
def calculate_zone_distribution(si_data):
    """
    Расчёт распределения времени по зонам
    """
    total = len(si_data)
    
    green = sum(1 for si in si_data if si < 150) / total * 100
    yellow = sum(1 for si in si_data if 150 <= si < 250) / total * 100
    red = sum(1 for si in si_data if si >= 250) / total * 100
    
    return {
        'green': round(green),
        'yellow': round(yellow),
        'red': round(red)
    }
```

```python
def analyze_trend(data):
    """
    Анализ общего тренда за период измерения
    """
    hr_data = data['Heart_Rate_bpm'].values
    si_data = data['Approximate_Stress_Index'].values
    
    # Делим на 4 квартиля
    q1, q2, q3, q4 = np.array_split(hr_data, 4)
    
    hr_trend = 'стабильный'
    if np.mean(q4) > np.mean(q1) * 1.1:
        hr_trend = 'нарастающий'
    elif np.mean(q4) < np.mean(q1) * 0.9:
        hr_trend = 'снижающийся'
    
    # Вариабельность HR
    hr_std = np.std(hr_data)
    if hr_std < 3:
        hr_variability = 'низкая'
    elif hr_std < 6:
        hr_variability = 'умеренная'
    else:
        hr_variability = 'высокая'
    
    # Количество всплесков
    events = detect_events(data)
    spikes_up = sum(1 for e in events if e['type'] == 'spike_up')
    spikes_down = sum(1 for e in events if e['type'] == 'spike_down')
    
    # Формирование вывода
    trend_text = f"HR: {hr_trend} тренд, {hr_variability} вариабельность (±{hr_std:.0f} уд/мин). "
    trend_text += f"Зафиксировано {spikes_up} эпизодов активации"
    if spikes_down > 0:
        trend_text += f" и {spikes_down} эпизодов снижения"
    trend_text += "."
    
    # Общий вывод
    if spikes_up <= 2 and hr_variability != 'высокая':
        conclusion = "Состояние стабильное, реактивность в норме."
    elif spikes_up > 4:
        conclusion = "Повышенная реактивность, частые всплески активации."
    else:
        conclusion = "Умеренная реактивность, периодические колебания состояния."
    
    return {
        'text': trend_text,
        'conclusion': conclusion,
        'hr_trend': hr_trend,
        'hr_variability': hr_variability,
        'spikes_up': spikes_up,
        'spikes_down': spikes_down
    }
```

```python
class NeuroHRVDashboard:

        'green_zone': '#00d4aa',
        'yellow_zone': '#ffd93d',
        'red_zone': '#ff4757'
    }
    
    def __init__(self, data, period_name="5 мин"):
        """
        data: pandas DataFrame с колонками:
            - Timestamp_ISO
            - Heart_Rate_bpm
            - RR_Interval_ms
            - Approximate_RMSSD
            - Approximate_SDNN
            - Approximate_Stress_Index
        """
        self.data = data
        self.period_name = period_name
        self.metrics = self._calculate_all_metrics()
        self.axis_scores = self._calculate_axis_scores()
        self.events = detect_events(data)
    
    def _calculate_all_metrics(self):
        """Расчёт всех HRV метрик"""
        rr = self.data['RR_Interval_ms'].values
        hr = self.data['Heart_Rate_bpm'].values
        
        # Базовые метрики
        mean_rr = np.mean(rr)
        mean_hr = np.mean(hr)
        sdnn = np.std(rr, ddof=1)
        rmssd = np.sqrt(np.mean(np.diff(rr)**2))
        
        # pNN50
        diff_rr = np.abs(np.diff(rr))
        pnn50 = (np.sum(diff_rr > 50) / len(diff_rr)) * 100
        
        # Poincaré
        sd1 = np.sqrt(0.5 * rmssd**2)
        sd2_sq = 2 * sdnn**2 - 0.5 * rmssd**2
        sd2 = np.sqrt(max(0, sd2_sq))
        
        # Stress Index (из данных или расчётный)
        if 'Approximate_Stress_Index' in self.data.columns:
            stress_index = np.mean(self.data['Approximate_Stress_Index'].values)
        else:
            stress_index = self._calculate_stress_index(rr)
        
        # Частотный анализ (упрощённый)
        lf_power, hf_power = self._estimate_frequency_powers(rr)
        lf_hf_ratio = lf_power / hf_power if hf_power > 0 else 1.0
        
        # SNS и PNS
        total_power = lf_power + hf_power
        sns_percent = (lf_power / total_power * 100) if total_power > 0 else 50
        pns_percent = (hf_power / total_power * 100) if total_power > 0 else 50
        
        # Частота дыхания
        respiratory_rate = self._estimate_respiratory_rate(rr)
        
        # Биологический возраст
        biological_age = self._calculate_biological_age(rmssd, sdnn, stress_index)
        
        return {
            'mean_rr': mean_rr,
            'mean_hr': mean_hr,
            'sdnn': sdnn,
            'rmssd': rmssd,
            'pnn50': pnn50,
            'sd1': sd1,
            'sd2': sd2,
            'stress_index': stress_index,
            'lf_power': lf_power,
            'hf_power': hf_power,
            'lf_hf_ratio': lf_hf_ratio,
            'sns_percent': sns_percent,
            'pns_percent': pns_percent,
            'respiratory_rate': respiratory_rate,
            'biological_age': biological_age
        }
```
