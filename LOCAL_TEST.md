# 🧪 Тестирование на локальном ноутбуке

## ✅ Бот запущен локально

Бот работает на вашем ноутбуке и готов к тестированию.

## 🎯 Варианты тестирования

### Вариант 1: Отправка CSV файла (самый простой)

1. Откройте Telegram: `@HRV_HR_UP_bot`
2. Отправьте `/start`
3. Отправьте CSV файл (например, `hrv_export_5min_20251010_172946.csv` из папки)
4. Используйте `/analyze` - получите дашборды

### Вариант 2: Прямое подключение к Polar (если устройство рядом)

1. `/scan` - найти Polar устройство
2. `/connect` - подключиться
3. `/start_measurement` - начать измерение (5 минут)
4. `/analyze` - получить дашборды

**Примечание:** Работает только если Polar устройство физически рядом с ноутбуком (Bluetooth диапазон).

## 📊 Проверка работы

### Команды для проверки:

- `/start` - приветствие
- `/help` - справка
- `/status` - статус (если подключено к Polar)

### Логи бота:

```bash
tail -f logs/bot.log
```
