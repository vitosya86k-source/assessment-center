# Neiry Mind Tracker — ресёрч и план добавок (кружочки + live-АЦ)

## Зачем
Neiry Mind Tracker (сухоэлектродный ЭЭГ-ободок/наушники, 4 канала, ритмы альфа/бета/тета) брали на тест. **Боли подтверждены пользователем:**
- на длинных/плотных волосах плохой контакт → периодически отпадает, плохой импульсный сигнал;
- 8 часов носить нельзя;
- админка убогая: ~5 метрик, в интерфейсе выбираются только 3, остальные не отображаются; «бета для прошаренных» не открылась даже у программистов;
- **не сохраняет данные по 15-мин/8-час окнам** → результаты теста сильно «гуляют».

**Наш путь решает всё это:** камера+звук, без железа, без проблемы волос, **пишем все каналы непрерывно на диск**, метрик уже больше + соц-оси.

## Что меряет Neiry (для справки)
3 блока: **когнитивная нагрузка** (0–100%), **состояние** (вовлечённость/расслабление/тревога/стресс/утомление/выгорание), **эмоц. статус** (сфокусирован/спокоен/напряжён). Плюс flow, iAF-профиль (креатив vs концентрация), рекомендации работа-отдых, нейрофидбек.
База: ритмы + iAF Базановой + индекс утомления Климеша. Валидировано в основном *качество сигнала* (Лебедев, bioRxiv), не сами метрики.

## Карта: метрика Neiry → наш прокси по камере/звуку → реалистичность
| Метрика Neiry | Наш прокси | Куда | Реалистичность |
|---|---|---|---|
| Стресс / тревога | ↑ЧСС + ↑/поверхностное дыхание + напряжение мимики + ↑возбуждение/↓валентность + сжатость голоса | оба | **высокая** |
| Расслабление | ровный пульс, медленное дыхание, нейтрально-позитивная валентность, расслабл. поза | оба | высокая |
| Возбуждение/валентность | прямой выход HSEmotion + пульс/дыхание | оба | высокая |
| Утомление | частота/длительность морганий (PERCLOS) + оседание позы/головы + замедление речи + длинные паузы + монотонность | оба | средняя |
| Вовлечённость/фокус | направление взгляда + наклон вперёд + неподвижность головы | АЦ | средняя (поведенческий) |
| Когнитивная нагрузка | моргания + рост пауз/филлеров + напряжение лба | АЦ | средняя (поведенческий) |
| Выгорание (тренд) | лонгитюд: накопление стресс+утомление по дням | кружочки | средняя |
| iAF, flow, ЭЭГ-индекс нагрузки, нейрофидбек | — | — | **только с ЭЭГ** |

**Наше преимущество над Neiry:** оси Тодорова (доверие/доминантность) + речевые маркеры — у них этого нет.

## ПЛАН ДОБАВОК

### A. Композитные индексы (собрать из того, что уже считается)
1. **Стресс-индекс** (0–100): нормировать и сложить ЧСС↑ + дыхание↑ + напряжение мимики (brow/mouth blendshapes) + возбуждение + интонационная нестабильность. Подсветка зон.
2. **Утомление** (0–100): моргания (PERCLOS, через FaceLandmarker — добавить детект глаз) + оседание позы/головы + замедление речи + длинные паузы + монотонность интонации.
3. **Вовлечённость/фокус** (поведенческий, метить явно): взгляд на экран + наклон вперёд + неподвижность.

### B. Подытоживающие блоки (ВЫВОДЫ — главное по запросу)
Сверху панели 3–4 человекочитаемые карточки-вывода, синтез из метрик:
- **Состояние сейчас:** напр. «Напряжён, стресс растёт» / «Спокоен, собран».
- **Динамика:** «К концу устаёт — моргает чаще, речь замедлилась».
- **Как считывается (Тодоров):** «Настороженно, держит дистанцию».
- **Речь:** «Логично, но монотонно» (live-прокси / точнее по диктофону).
Детальные метрики — ниже, сворачиваемо.

### C. Для кружочков
- 3-блок подача (состояние/эмоции/возбуждение-спокойствие);
- **тренд выгорания/восстановления** линией по дням;
- мягкие рекомендации работа-отдых;
- окна усреднения (минута/час/день) — сгладит шумный rPPG.

## Чего честно НЕ сделаем без ЭЭГ
iAF и «когнитивный профиль», истинный flow, когнитивная нагрузка как ЭЭГ-индекс, нейрофидбек по ритмам, ментальное vs физическое возбуждение на уровне ЦНС.
