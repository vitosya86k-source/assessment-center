# 🚀 Быстрый старт

## Установка и запуск

### 1. Установка зависимостей

```bash
cd "NEW VERSION"
pip install -r requirements.txt
```

### 2. Проверка работы калькулятора

```bash
python test_calculator.py
```

Это создаст тестовые данные и проверит работу всех модулей.

### 3. Запуск бота

```bash
python run_bot.py
```

Или напрямую:

```bash
python telegram_bot.py
```

## 📱 Использование

1. Найдите бота в Telegram: `@HRV_HR_UP_bot`
2. Отправьте команду `/start`
3. Отправьте CSV файл с данными HRV (можно использовать пример из папки)
4. Используйте `/analyze` для получения дашбордов

## 📊 Формат данных

CSV файл должен содержать минимум колонку `Heart_Rate_bpm`:

```csv
Heart_Rate_bpm,Timestamp_ISO
75,2025-01-22 17:24:45
76,2025-01-22 17:24:46
...
```

## 🔧 Интеграция с Android

### Вариант 1: Через файлы

1. Android приложение сохраняет CSV в папку
2. Синхронизируйте файлы на сервер (ADB, облако, локальная сеть)
3. Бот читает из `android_sync/`

### Вариант 2: Через API (требует доработки)

Android приложение отправляет данные на API endpoint, бот получает через API.

## ⚠️ Важно

- Данные приближенные (RR рассчитывается из HR)
- Минимум 5 минут данных для корректного анализа
- Токен бота уже настроен в `config.py`

## 📁 Структура

- `data/` - загруженные пользователями данные
- `dashboards/` - сгенерированные PNG дашборды
- `exports/` - экспортированные CSV файлы
- `logs/` - логи бота
