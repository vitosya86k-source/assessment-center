# Контент-анализ речи Романа — таблица параметров
**Что это:** механика оценки речи. Левая колонка — параметр и категория. Средние колонки — значения по упражнениям. Правая — что это значит, согласно психолингвистическим best practices, со ссылкой на источник.
**Источники данных:** openSMILE eGeMAPS, parselmouth (Praat), pymorphy2 (русский POS-tagger), ruBERT-tiny2-cedr (русские эмоции), POS-rPPG из видео.
**Транскрипт:** whisper-small (medium точнее для русского, small шумный). Кластеризация по медиане F0 (Роман = male_low <140 Hz).
**Колонки = упражнения, в которых Роман присутствовал и говорил:** письма, сотрудник, клиент.
**«Разбор» НЕ включён** — это упражнение асессоры обсуждали Романа в его отсутствие. Их слова о нём см. в `roman_portrait_v2.md`.
**Длительность речи Романа:** письма ~295 с / сотрудник ~104 с / клиент ~335 с.

## Краткая синтез-картина


**Голос:** узкая интонация (F0 IQR 5.9-11.2 vs норма 15+), нормальная динамика громкости, низкий pitch IQR. **Стиль «со всеми одинаково ровно»** — асессоры назвали это так в разборе. Не зажат, а **стилево не играет голосом**.

**Темп:** 80-113 wpm — **обдумывающий** (норма 130-150). Паузы 11-16 с в среднем, максимум 40-59 с. Долгих пауз (>2 с) — 87-92% всех. **Большая часть его речи — обдумывание, не реакция.**

**Лексика:** существительные 19-28% (предметность), глаголы 10-12% (низковато — не «делатель»), длинные слова 35-47%. TTR падает с 0.69 в письмах до 0.51 в клиенте = **когнитивное сжатие под стрессом**.

**Местоимения:** I/We ratio = 7 (письма) → 4 (сотрудник) → 0.9 (клиент). Он **переключается между ролями**: «я-эксперт» в письмах, «вы-должны» в сотруднике, «мы-партнёр» в клиенте. Гибкость регистра.

**Agency/Communion:** ~0.3-1.0 — communion-слова присутствуют, но это **формальные** ("команда", "сотрудничество"), не настоящая ориентация на людей (асессоры это заметили).

**Need/Can:** 2.0 в письмах (нормативно-педагогический), 0.2 в клиенте (договаривающийся). **Стиль гибко меняется по задаче.**

**Эмоции ruBERT:** письма 97% no_emotion (полный аналитик), сотрудник 92% (контроль), **клиент 51%** + 19% anger + 15% joy = **«агрессивный энтузиазм продавца»**. Единственная ситуация, где он эмоционально включается.

**Coping:** стабильно ~50% problem + ~50% avoidant + ~0% emotion. **Защита через логику + хеджирование**, не через выражение чувств. Кроме «сотрудника» — там впервые 33% emotion (вынужденно).

**Физиология:** HR 60 в 36-мин клиенте (парасимпатика доминирует, **trait anxiety отсутствует**). Intensity hub 0.04-0.09 — нет хронической тревоги. Алексей утром был 2.4.

## Что дальше для production механики

3. **Популяционная норма** — при N=10-20 АЦ-участников через стек строим квартили для каждого параметра.
