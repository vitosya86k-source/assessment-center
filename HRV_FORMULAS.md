# 📐 Формулы расчета HRV показателей

## ⚠️ ВАЖНО: Данные приближенные
RR интервалы рассчитываются из HR: `RR (мс) = 60000 / HR (уд/мин)`

---

## 1. Базовые временные метрики

### SDNN (Standard Deviation of NN intervals)
```
SDNN = √(Σ(RRᵢ - mean_RR)² / (n-1))
```
**Что показывает:** Общая вариабельность сердечного ритма

### RMSSD (Root Mean Square of Successive Differences)
```
RMSSD = √(Σ(RRᵢ₊₁ - RRᵢ)² / (n-1))
```
**Что показывает:** Парасимпатическая активность (краткосрочная вариабельность)

### pNN50 (Percentage of NN intervals differing by more than 50ms)
```
pNN50 = (количество |RRᵢ₊₁ - RRᵢ| > 50 мс) / (n-1) × 100%
```
**Что показывает:** Гибкость нервной системы

---

## 2. Метрики Пуанкаре

### SD1 (Short-term variability)
```
SD1 = √(0.5 × var(diff_RR))
где diff_RR = [RR₂-RR₁, RR₃-RR₂, ..., RRₙ-RRₙ₋₁]
```
**Что показывает:** Краткосрочная вариабельность (парасимпатическая активность)

### SD2 (Long-term variability)
```
SD2 = √(2 × var(RR) - 0.5 × var(diff_RR))
```
**Что показывает:** Долгосрочная вариабельность (выносливость)

---

## 3. Stress Index (Индекс напряжения по Баевскому)

```
SI = AMo / (2 × Mo × MxDMn)

где:
- AMo = (количество RR в модальном классе) / (общее количество RR) × 100%
- Mo = центр бина с максимальной частотой (из гистограммы с 50 бинами)
- MxDMn = max(RR) - min(RR) (вариационный размах)

Mo и MxDMn в секундах для расчета: Mo/1000, MxDMn/1000
```

**Диапазоны:**
- <50: Расслабление
- 50-150: Норма
- 150-250: Напряжение
- 250-350: Выраженное напряжение
- >350: Перенапряжение

---

## 4. Частотные метрики

### LF Power (Low Frequency, 0.04-0.15 Гц)
```
1. Интерполяция RR к равномерной сетке (fs = 4 Гц)
2. Удаление тренда
3. Спектральный анализ (Welch method)
4. Интеграция мощности в диапазоне 0.04-0.15 Гц
```

### HF Power (High Frequency, 0.15-0.4 Гц)
```
Аналогично LF, но интеграция в диапазоне 0.15-0.4 Гц
```

### LF/HF Ratio
```
LF/HF = LF_power / HF_power
```

**Диапазоны:**
- <0.5: ПНС доминирует
- 0.5-2.0: Оптимальный баланс
- 2.0-4.0: СНС умеренно
- >4.0: Стресс

---

## 5. Показатели 7 осей паутинки

### RD — Readiness (Готовность)
```
Нормализация:
- rmssd_norm = min(rmssd / 50 * 100, 100)  # 50 мс = 100%
- sdnn_norm = min(sdnn / 100 * 100, 100)   # 100 мс = 100%
- si_norm = max(0, 100 - (stress_index / 5))  # SI 500 = 0%
- hf_norm = min(hf_power / 500 * 100, 100)  # 500 ms² = 100%

LF/HF баланс:
- если 0.5 <= lf_hf_ratio <= 2.0: balance_norm = 100
- если lf_hf_ratio < 0.5: balance_norm = lf_hf_ratio / 0.5 * 80 + 20
- если lf_hf_ratio > 2.0: balance_norm = max(0, 100 - (lf_hf_ratio - 2) * 20)

RD = 0.30 × rmssd_norm + 0.25 × sdnn_norm + 0.20 × si_norm + 0.15 × hf_norm + 0.10 × balance_norm
```

### SR — Stress Resistance (Стрессоустойчивость)
```
SR% = f(SI)

если SI <= 50: SR = 100
если 50 < SI <= 150: SR = 100 - (SI - 50) * 0.25  # 50→100%, 150→75%
если 150 < SI <= 300: SR = 75 - (SI - 150) * 0.33  # 150→75%, 300→25%
если 300 < SI <= 500: SR = 25 - (SI - 300) * 0.10  # 300→25%, 500→5%
если SI > 500: SR = max(0, 5 - (SI - 500) * 0.01)
```

### AD — Adaptability (Адаптивность)
```
AD% = f(SDNN)

если SDNN >= 100: AD = 100
если 50 <= SDNN < 100: AD = 50 + (SDNN - 50) * 1.0  # 50→50%, 100→100%
если 30 <= SDNN < 50: AD = 30 + (SDNN - 30) * 1.0  # 30→30%, 50→50%
если 20 <= SDNN < 30: AD = 15 + (SDNN - 20) * 1.5  # 20→15%, 30→30%
если SDNN < 20: AD = max(0, SDNN * 0.75)  # 0→0%, 20→15%
```

### FL — Flexibility (Гибкость НС)
```
Нормализация:
- pnn50_norm = min(pnn50 / 20 * 100, 100)  # 20% = 100%
- sd1_norm = min(sd1 / 50 * 100, 100)  # 50 мс = 100%

FL = 0.6 × pnn50_norm + 0.4 × sd1_norm
```

### RC — Recovery (Восстановление)
```
Нормализация:
- rmssd_norm = min(rmssd / 60 * 100, 100)  # 60 мс = 100%
- hf_norm = min(hf_power / 1000 * 100, 100)  # 1000 ms² = 100%

RC = 0.5 × rmssd_norm + 0.5 × hf_norm
```

### EN — Endurance (Выносливость)
```
EN% = f(SD2)

если SD2 >= 100: EN = 100
если 70 <= SD2 < 100: EN = 70 + (SD2 - 70) * 1.0  # 70→70%, 100→100%
если 50 <= SD2 < 70: EN = 50 + (SD2 - 50) * 1.0  # 50→50%, 70→70%
если 30 <= SD2 < 50: EN = 30 + (SD2 - 30) * 1.0  # 30→30%, 50→50%
если SD2 < 30: EN = max(0, SD2)  # 0→0%, 30→30%
```

### BL — Balance (Вегетативный баланс)
```
BL% = f(LF/HF)

если 0.5 <= LF/HF <= 2.0: BL = 100  # Оптимальный баланс
если LF/HF < 0.5: BL = max(50, 50 + LF/HF * 100)  # 0→50%, 0.5→100%
если 2.0 < LF/HF <= 4.0: BL = max(40, 100 - (LF/HF - 2) * 30)  # 2→100%, 4→40%
если LF/HF > 4.0: BL = max(0, 40 - (LF/HF - 4) * 10)  # 4→40%, 8→0%
```

**ПРОБЛЕМА:** При LF/HF = 11.07:
- Старая формула: BL = max(0, 40 - (11.07 - 4) * 10) = max(0, -30.7) = 0% ❌
- Новая формула: BL = max(0, 40 - (11.07 - 4) * 5) = max(0, 4.65) = 4.65% ≈ 5% ✅

**ИСПРАВЛЕНИЕ:** Более плавное снижение (коэффициент 5 вместо 10)

---

## 6. Общий балл

```
Overall = 0.20 × RD + 0.20 × SR + 0.15 × AD + 0.10 × FL + 0.15 × RC + 0.10 × EN + 0.10 × BL
```

## 7. Биологический возраст

```
Базовая оценка через RMSSD:
- если rmssd >= 50: base_age = 20
- если 40 <= rmssd < 50: base_age = 20 + (50 - rmssd) * 1.0
- если 30 <= rmssd < 40: base_age = 30 + (40 - rmssd) * 1.5
- если 20 <= rmssd < 30: base_age = 45 + (30 - rmssd) * 1.5
- если 10 <= rmssd < 20: base_age = 60 + (20 - rmssd) * 1.5
- если rmssd < 10: base_age = 75 + (10 - rmssd) * 2.0

Коррекция по SDNN:
- если SDNN > 100: base_age -= 5
- если SDNN < 30: base_age += 5

Коррекция по Stress Index:
- если SI < 100: base_age -= 3
- если SI > 300: base_age += 5

biological_age = max(18, min(90, round(base_age)))
```

## 8. Частота дыхания

```
1. Спектральный анализ RR (как для HF)
2. Найти пик в диапазоне 0.15-0.4 Гц
3. respiratory_rate = peak_freq * 60  # Гц → вд/мин
4. Ограничить: max(8.0, min(25.0, respiratory_rate))
```

## 🔴 ПРОБЛЕМЫ В ТЕКУЩЕЙ РЕАЛИЗАЦИИ

### 1. BL (Баланс) при высоком LF/HF
**Проблема:** При LF/HF = 11.07 получается 0%, что слишком резко.

**Исправление:**
```
если LF/HF > 4.0: 
    BL = max(0, 40 - (LF/HF - 4) * 5)  # Более плавное снижение
    # 4→40%, 6→30%, 8→20%, 10→10%, 12→0%
```

### 2. RC (Восстановление) при низком RMSSD
**Проблема:** При RMSSD = 46.3 мс и низком HF может быть занижен.

**Проверка:** 
- rmssd_norm = 46.3/60*100 = 77%
- Если hf_norm низкий, RC будет низким

### 3. FL (Гибкость) при pNN50 = 0%
**Проблема:** pNN50 = 0% дает 0% в формуле, даже если SD1 нормальный.

**Проверка:** Формула правильная, но нужно проверить расчет pNN50.
