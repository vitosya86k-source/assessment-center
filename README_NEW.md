# NeuroHRV Monitor Bot — Новая версия

Telegram бот для анализа вариабельности сердечного ритма (HRV) с генерацией дашбордов и экспортом данных.

## 🎯 Возможности

- 📊 **Анализ HRV данных** — расчет 7 показателей паутинки (RD, SR, AD, FL, RC, EN, BL)
- 📈 **Генерация дашбордов** — 4 PNG изображения:
  - Профиль (паутинка + прогресс-бары)
  - SWOT-анализ (сильные стороны, ограничения, возможности, риски)
  - Динамика (лента состояния, график HR, события)
  - Памятка (расшифровка показателей)
- 📤 **Экспорт данных** — CSV файлы с метриками
- 📱 **Интеграция с Polar** — через Android приложение или CSV файлы

## 📋 Требования

- Python 3.10+
- Telegram Bot Token
- Установленные зависимости (см. requirements.txt)

## 🚀 Установка

1. Клонируйте репозиторий или скопируйте файлы
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте токен бота в `config.py`:
```python
TELEGRAM_BOT_TOKEN = "ваш_токен_здесь"
```

## 💻 Запуск

```bash
python telegram_bot.py
```

## 📱 Использование

### Команды бота:

- `/start` — начать работу с ботом
- `/help` — справка по командам
- `/analyze` — запустить анализ HRV данных
- `/export` — экспортировать данные в CSV

### Отправка данных:

1. Подготовьте CSV файл с данными HRV
2. Обязательные колонки:
   - `Heart_Rate_bpm` — частота пульса (уд/мин)
   - `Timestamp_ISO` — временные метки (опционально, создаются автоматически)
3. Отправьте файл боту
4. Используйте `/analyze` для получения дашбордов

### Формат CSV файла:

```csv
Second,Timestamp_ISO,Heart_Rate_bpm,RR_Interval_ms
0,2025-01-22 17:24:45,75,800.0
1,2025-01-22 17:24:46,76,789.5
...
```

## 🔧 Интеграция с Android приложением

Для подключения Polar устройства через Android приложение:

1. **Вариант A: Синхронизация через файлы**
   - Android приложение сохраняет данные в CSV
   - Файлы синхронизируются с сервером (через ADB, облако или локальную сеть)
   - Бот читает файлы из директории `android_sync/`

2. **Вариант B: API интеграция**
   - Android приложение отправляет данные на API endpoint
   - Бот получает данные через API (требует доработки)

3. **Вариант C: Прямая интеграция**
   - Использование библиотеки для работы с Polar (например, `polar-sdk`)
   - Требует дополнительной настройки Bluetooth

## 📊 Показатели HRV

### 7 осей паутинки:

- **RD** — Readiness (Готовность)
- **SR** — Stress Resistance (Стрессоустойчивость)
- **AD** — Adaptability (Адаптивность)
- **FL** — Flexibility (Гибкость НС)
- **RC** — Recovery (Восстановление)
- **EN** — Endurance (Выносливость)
- **BL** — Balance (Вегетативный баланс)

### Дополнительные показатели:

- **HRV** — вариабельность сердечного ритма (SDNN)
- **Stress Index** — индекс напряжения по Баевскому
- **HR** — частота сердечных сокращений
- **LF/HF** — соотношение симпатической и парасимпатической активности
- **Дыхание** — частота дыхания
- **Биологический возраст** — функциональный возраст по HRV

## ⚠️ Важные замечания

1. **Приближенные данные**: RR интервалы рассчитываются из HR как `RR = 60000 / HR`. 
   Для точных измерений нужен датчик с реальными RR интервалами.

2. **Минимальная длительность**: Рекомендуется минимум 5 минут данных для корректного анализа.

3. **Частотный анализ**: Требует достаточного количества данных (минимум 60 секунд).
