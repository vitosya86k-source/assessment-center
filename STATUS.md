### Интеграция с Android:

**Вам НЕ нужно дорабатывать Android приложение!**

Просто:
1. Android приложение сохраняет CSV файлы
2. Синхронизируйте файлы в папку `android_sync/` (через ADB, облако или сеть)
3. Бот автоматически найдет и обработает файлы

Подробнее: см. `ANDROID_INTEGRATION.md`

### Управление ботом:

**Остановить:**
```bash
pkill -f "run_bot.py"
```

**Перезапустить:**
```bash
cd "/home/user/с винды/HRV_Monitor_bot/NEW VERSION"
./venv/bin/python run_bot.py
```

**Или используйте скрипт:**
```bash
./start_bot.sh
```
